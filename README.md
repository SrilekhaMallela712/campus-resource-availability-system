# campus-resource-availability-system
Core Java project to track campus resource availability
A Java-based academic project that demonstrates how campus resources such as classrooms and laboratories can be tracked and displayed using core Java concepts.
About the Project:
The Campus Resource Availability System is designed to provide a simple way to check the availability of campus resources.
This repository focuses on demonstrating basic Java programming skills and structured problem-solving in an academic context.
Features:
Display available campus resources
Check resource availability status
Simple and beginner-friendly implementation
Uses core Java programming concepts
How to Run:
Make sure Java is installed on your system.
Clone or download this repository.
Open the project in any Java IDE (Eclipse, IntelliJ IDEA, or NetBeans).
Run the main Java file.
View the output in the console or application window.
Technologies Used:
Java
Core Java concepts (OOP, control structures)
